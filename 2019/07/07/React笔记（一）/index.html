React是什么？官网是这么说的：“React 是一个声明式，高效且灵活的用于构建用户界面的 JavaScript 库。使用 React 可以将一些简短、独立的代码片段组合成复杂的 UI 界面，这些代码片段被称作‘组件’。”

  3. 虚拟DOM

    React会在内存中维护一个虚拟DOM树，当我们对这个树进行读或者写的时候实际上是对虚拟DOM进行的。

    当数据变化时，React会自动更新虚拟DOM，然后拿新的虚拟DOM和旧的虚拟DOM进行对比，找到有变更的部分，得出一个Patch，然后将这个Patch放到一个队列里面，最终批量更新这些Patch到DOM中。

    这样的机制可以保证即便是根节点数据的变化，最终表现在DOM上的修改也只是收这个数据影响的部分，这样可以保证非常高效地渲染。
