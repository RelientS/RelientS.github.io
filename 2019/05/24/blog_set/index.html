不得不说，前端真是一些爱折腾的人。什么东西都想搞搞，之前先是用着wordpress，感觉太low，还需要后端搭建一个数据库。后来又用Vuepress，但是主题太少了，不想千篇一律。最后看来看去，还是用HEXO吧。
