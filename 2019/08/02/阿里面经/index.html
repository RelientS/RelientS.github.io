阿里面经

发表于 2019-08-02 | 更新于 2019-08-01 | 分类于 前端技术 ， 面经

又和阿里面试官谈笑风生了~

来记录一下这两次面试都问了些啥吧。

一面

一面其实我以为不是面试。。。面试官就和我说，随便聊聊，不问技术问题。

不过其实也确实没问啥技术问题，简单记录一下。

就问了问虚拟DOM和Diff算法。

然后问了问项目，提了点建议，希望我把知识点系统化，就是把知识串起来。这点其实在二面中也有体现，或者说在各大公司面试中，都是很重要的一个思路。

提问环节：我问了一下加班、未来发展等等。还问了一个问题：5G给前端带来了什么机遇，或者说前端会怎么发展。

面试官：服务端渲染->移动端只是一个屏幕->大部分逻辑都在服务端完成

最后给我的建议：结合业务，在业务里面实践新技术，这点其实在头条也能感受到。

二面

二面算是正式的技术面了，一面的“小弟”加班到一半来面的我，被我吐槽了一会儿。

全程有说有笑，真的不太像是在面试。

上来先来个自我介绍吧：练习时长一年半的前端实习生，擅长唱跳rap篮球balahbalah。

  1. 先问了问，面向过程和面向对象的区别（这啥，下马威？

  2. 然后来了一道经典题：从输入url开始到页面渲染的整个过程，每个步骤都展开细问

    1、url包括哪些部分，DNS解析过程

    2、说到域名，讲讲跨域的四种方式？JSONP的原理知道吗？除了script标签还有啥办法？

    3、该发请求啦，说说TCP和UDP的区别，要用TCP实现一个聊天软件，怎么能达到UDP的速度（答不出来

    4、简单聊一聊HTTP 1/1.1/2.0

    5、OK，后端把东西返回来了，浏览器如何渲染呢？DOM+CSSOM

  3. 接着到了JS环节，问了两题

    1、用generator实现一个async。

    2、说一下js事件循环。setTimeout(xxx,0)是立刻执行吗？Promise、setTimeout、console.log执行顺序

  4. 接着提问，该到MVVM框架了。问些啥呢，说一下VDOM和Diff算法吧（真就喜欢问这个

  5. 设计模式：说一说单例模式和工厂模式吧，让你用js实现一个单例怎么做（闭包）。

  6. 数据结构（您放过我吧），我说我只会链表和树，还是二叉树。面试官：行吧，判断链表有环怎么做。（快慢指针）我：谢谢您！

  7. 问完基础问题，该问项目了。问了一下项目的选型，socket.io的广播和组播啥的。

到这里差不多就结束了，两个人聊开心了，面了一个多小时左右。所以最后到了提问环节，我皮了一下，说有个主观题你回答一下，面试官：你是不是不想进了哈哈哈哈。

问题还是5G的远望，面试官：主要在于多媒体方面吧，比如说在线游戏这些，浏览器上玩王者荣耀？也是个有趣的思路~

最后给我的建议：找时间补数据结构基础，多在公司项目里想想，后端的数据结构是不是最优了，能不能再进行优化，或者更适合前端使用？还有平时要积累算法，“等到你想补算法的时候就迟了。”

未完待续~
